字体 #

控制元素字体的工具类。

使用方法 #

sans 字体serif 字体mono 字体
html
<span class="sans">sans 字体</span>
<span class="serif">serif 字体</span>
<span class="mono">mono 字体</span>
1
2
3

使用 sans 可应用一个网络安全的无衬线字体；使用 serif 可应用网络安全的衬线字体；使用 mono 可应用一个网络安全的等宽字体。

seriffont-family: ui-serif, Georgia, Cambria, "Times New Roman", Times, serif;
monofont-family: ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace;
