CSS 类 ​

面板提供了如下 CSS 类：

类类型作用
panel实体类元素作为面板组件
panel-heading实体类元素作为面板头部组件
panel-title实体类元素作为面板头部标题组件
panel-body实体类元素作为面板主体组件
panel-footer实体类元素作为面板尾部组件
